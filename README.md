# PAGviz
PAGviz is a simple command-line interface for displaying Partial Ancestral Graphs (PAGs) or Directed Acyclic Graphs (DAGs) exported in a text format from the [Tetrad](https://github.com/cmu-phil/tetrad) program for causal discovery.

It can output the graph either as an image (`.png` by default) generated by the [pydot](https://github.com/pydot/pydot) Python package, or a `.dot` file that can be edited and displayed separately using any Graphviz-compatible software. The image can also be sent to `stdout` as an option (tested on a Linux machine).

## Usage
```
PAGviz [-h] [-w] [-d] [--format FORMAT] [-f FILE_OUTPUT] [-t] input_file

positional arguments:
  input_file

options:
  -h, --help            show this help message and exit
  -w, --white-background
                        Add a white background to the graph instead of OldLace
  -d, --dot-file        Produce a dot file instead of an image
  --format FORMAT       The format of the image file if you want to use another format than the default PNG. Example: 'svg'

output:
  Choose one or more output modes

  -f, --file-output FILE_OUTPUT
                        A path with a filename (without the extension) to save the output to a desired location
  -t, --terminal-output
                        Output the result to stdout
```
